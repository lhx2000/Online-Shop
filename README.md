# Stream使用指南

在本地运行后在相应端口输入‘/SignUp’进入注册界面
注册成功后点击右上角的登录选项进行登录，成功登录即可体验商城各项功能

# Stream路由总结

注册：/SignUp
登录：/SignIn
主页：/HomePage
个人中心：/Account
搜索列表：/SearchList
商品页面：/Game?GID=G2020XXXXX
